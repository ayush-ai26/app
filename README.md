# Healthcare AI Assistant

AI-powered healthcare assistant mobile app prototype designed for symptom support, emergency accessibility, and doctor connectivity.

## Features

* AI symptom checker
* Emergency help section
* Doctor consultation interface
* Anonymous support mode
* Authentication system
* Mobile app interface

## Technologies Used

* React Native
* Expo
* Backend APIs
* AI chatbot integration

## Disclaimer

This application is a student-built prototype intended for educational purposes only and is not a substitute for professional medical advice.
